# 🎧 Spotify Listening Analytics Dashboard

A modern Spotify-inspired Power BI dashboard that transforms raw streaming history into actionable insights. This project analyzes listening behavior, artist performance, platform usage, albums, and songs using Excel for data cleaning and Power BI for visualization.

---

##  Project Overview

This dashboard provides an interactive analysis of Spotify streaming history to identify listening patterns, favorite artists, platform preferences, and music engagement over time.

The project follows a complete data analytics workflow:

- Data Cleaning (Excel)
- Data Analysis
- Data Modeling
- DAX Measures
- Interactive Dashboard Design
- Business Insights

---

## 🛠 Tools Used

- Microsoft Excel
- Power BI
- DAX
- Power Query

---

## 📂 Dataset Information

The dataset contains Spotify streaming history with the following fields:

- Spotify Track URI
- Date
- Month Name
- Time Stamp
- Platform
- Milliseconds Played
- Minutes Played
- Track Name
- Artist Name
- Album Name
- Reason Start
- Reason End
- Shuffle
- Skipped
- Played vs Skipped

- Total Songs Played
- Total Artists
- Total Albums

- Total Listening Hours
- Active Listening Days
- Platforms Used
- Maximum Play Time

**Visuals**

- Monthly Listening Trend
- Top Artists
- Platform Distribution

### 🎤 Artist Analysis

Analyze artist performance through:

### 🎵 Song Analysis

- Top Tracks
